Original Recipe:



Hacker Dumplings V1.0a 

1kg Baby Bok Choy

1kg Ground Pork (30-50% fat)

90g Green Onion

45g (3 tbsp.) Soy Sauce

45g (3 tbsp.) Cooking Wine

30g Garlic

10g Ginger

5g (1 tsp.) Salt

5g (1 tsp.) Sugar

1 Egg

120 Dumpling Wrappers

32g (1/4 cup) Flour
